
+++++++++++++++++++++++++++++++++++++++
 Opengl
+++++++++++++++++++++++++++++++++++++++

- OpenGL 1.0-1.5（OpenGL 2.0版本之前），固定管线
- OpenGL 2.0-2.1（OpenGL 2.0版本系列），固定管线和可编程管线并存（2004.09）
- OpenGL 3.0-4.x（OpenGL 3.0版本之后），可编程管线崛起（2008.08）



====================================================================================================
 OpenGL 1.0
----------------------------------------------------------------------------------------------------
1992.01



====================================================================================================
  OpenGL 1.1
----------------------------------------------------------------------------------------------------
1997.01

- Vertex arrays取代了glVertex这类立即模式绘制函数，多个数据可以被一个函数调用绘制了，降低cpu开销。
- texture object的出现改变了过去只能使用显示表（display list）来静态使用纹理的方法。



====================================================================================================
  OpenGL 1.2
----------------------------------------------------------------------------------------------------
1998.03

- 某些扩展带来了重要的MipMap技术



====================================================================================================
  OpenGL 1.2.1
----------------------------------------------------------------------------------------------------
1998.10

- 专门介绍了ARB扩展的概念，这样的扩展将被广泛的实现。



====================================================================================================
  OpenGL 1.3
----------------------------------------------------------------------------------------------------
2001.08

- compressed texture，可以有效减少存储和带宽的压力，现在广泛用于各种对存储和带宽敏感的手持设备上。
- texture cube，主要用于skybox、动态反射（dynamic reflection）等技术上
- multisample，可以支持纹理和FrameBuffer的MSAA抗锯齿技术。



====================================================================================================
  OpenGL 1.4
----------------------------------------------------------------------------------------------------
2002.07

- 支持自动生成Mipmap



====================================================================================================
  OpenGL 1.5
----------------------------------------------------------------------------------------------------
2003.07

- buffer object，彻底取代了过去的vertex array和立即模式，顶点数据可以上传到显存了。




====================================================================================================
  OpenGL 2.0（!!!!!） ---> GLSL 110
----------------------------------------------------------------------------------------------------
2004.09

- GLSL出现（vertex shader和fragment shader），这个阶段出现了固定管线和可编程管线并存的情况。
- fragment shader可输出到帧缓冲的多个渲染目标（render target）上。

- 纹理不再有2^N的大小限制



====================================================================================================
  OpenGL 2.1 ---> GLSL 120
----------------------------------------------------------------------------------------------------
2006.07

- pixel buffer object，用来更快的传输像素




====================================================================================================
  OpenGL 3.0（!!!!!） ---> GLSL 130
----------------------------------------------------------------------------------------------------
2008.08

这个版本变化非常大，开始分core profile和compatibility profile

- 正式把帧缓存对象（framebuffer object）划入core profile，具有离线的帧缓冲了
- 帧缓冲对象之间可以互相拷贝像素到持有的不同的render target，性能上得到提升。

- 增加VAO，方便管理buffer object以及vertex attrib pointer及其开启状态，不必每次渲染前都要设置一遍了。（状态模式!!）

- 增加条件渲染（conditional rendering）。



====================================================================================================
  OpenGL 3.1 ---> GLSL 140
----------------------------------------------------------------------------------------------------
2009.03

- instanced rendering，减轻了同类物体绘制所占有的带宽压力。
- uniform buffer object



====================================================================================================
  OpenGL 3.2 ---> GLSL 150
----------------------------------------------------------------------------------------------------
2009.08

- geometry shader，可以用来生成新的图元类型（点、线、三角形）
- texture正式支持multisample，可以作为render target来进行framebuffer object上的抗锯齿




====================================================================================================
  OpenGL 3.3 ---> GLSL 330
----------------------------------------------------------------------------------------------------
2010.03

+++++++++++++++++++++++++++++++++++++++
 OpenGL ES
+++++++++++++++++++++++++++++++++++++++

- OpenGL ES 1.0是以OpenGL 1.3规范定义的
- OpenGL ES 1.1是以OpenGL 1.5规范定义的
- OpenGL ES 2.0是以OpenGL 2.0规范定义的（2007.03）

====================================================================================================
  OpenGL ES 2.0
----------------------------------------------------------------------------------------------------
2007.08，初代iphone和android内置了gles1.x






====================================================================================================
  OpenGL ES 2.0
----------------------------------------------------------------------------------------------------
2009.10，iphone3GS和android2.2内置了gles2.0







====================================================================================================
  OpenGL ES 3.0
----------------------------------------------------------------------------------------------------
2013.08，android4.3和ios7.0先后发布并支持


- 支持更多buffer object，增加Uniform Buffer Object（UBO）
- 支持32位整数和浮点数
- 增加多个纹理的支持: 包括浮点纹理、深度纹理、顶点纹理
- 多重渲染目标: 让GPU一次渲染多个纹理
- 多重采样抗锯齿（MSAA RTT），可提升图像效果
- 使用统一的纹理压缩格式ETC
